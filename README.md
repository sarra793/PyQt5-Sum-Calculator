# 🧮 PyQt5 Sum Calculator

A simple desktop application built with **Python** and **PyQt5** that performs the addition of two numbers.

## ✨ Features

- ➕ Add two numbers
- ✅ Input validation
- 📜 History of calculations
- 🔄 Reset all fields
- 🖥️ Easy-to-use graphical interface

## 📂 Project Structure

```
.
├── main.py
├── somme.ui
└── README.md
```

## 🚀 Requirements

- Python 3.x
- PyQt5

Install PyQt5:

```bash
pip install PyQt5
```

## ▶️ Run the Application

```bash
python main.py
```

- 🐍 Python

## 📖 How to Use

1. Enter the first number.
2. Enter the second number.
3. Click **Calculate**.
4. View the result.
5. All calculations are saved in the history.
6. Click **Reset** to clear everything.
